# Sistema de Estoque - Enfermagem

Sistema web para controle de materiais de enfermagem, desenvolvido para o SENAC Zona Norte sob responsabilidade da Enfermeira Camila.

## 🔗 Acesso

> [https://universidade-cesumar.github.io/prova-2bi-ads-esoft-3sem-Felipepiv/]

## 📋 Funcionalidades

- Cadastrar novos materiais com nome, quantidade e data
- Listar todos os materiais cadastrados
- Retirar quantidade do estoque
- Excluir materiais
- Pesquisar materiais por nome
- Dashboard com total de itens e alertas de estoque crítico (menos de 10 unidades)

## 🛠️ Tecnologias

- HTML, CSS e JavaScript puro
- API REST via MockAPI

## 📁 Estrutura

```
├── index.html
├── style.css
└── main.js
```

## ▶️ Como rodar

Basta abrir o arquivo `index.html` no navegador. Os dados são carregados automaticamente da API.
