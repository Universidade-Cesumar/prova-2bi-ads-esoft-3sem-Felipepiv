Sistema de Estoque — Enfermagem (SENAC ZN)
Sistema web simples para cadastrar e listar materiais de enfermagem do SENAC Zona Norte.

 Funcionalidades
Listar Materiais: Mostra os itens cadastrados direto da API.

Cadastrar Material: Formulário para adicionar nome e quantidade de novos insumos.

Design Responsivo: Funciona no celular e no computador.

 Tecnologias
HTML5

CSS3 (Grid e Flexbox)

JavaScript (Fetch API / Async-Await)

MockAPI (Simulação de banco de dados)

 Arquivos
index.html - Estrutura da página

style.css - Visual do sistema

script.js - Integração com a API
